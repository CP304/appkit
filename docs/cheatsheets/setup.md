# Setup
1) Flutter installieren.
2) `dart pub global activate melos`
3) Repo klonen → `melos bootstrap`
4) Android: Emulator/Device, USB‑Debugging an.
5) Start: `cd apps/app && flutter run`

Firebase (später):
- `google-services.json` nach `apps/app/android/app/`
- `GoogleService-Info.plist` nach `apps/app/ios/Runner/`
- `firebase_core` ist vorbereitet (auskommentiert).
