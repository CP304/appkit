# appkit (Flutter/Dart Monorepo)

Schnellstarter für neue Apps: Melos‑Monorepo, Riverpod, Dio, Drift, Firebase‑Hooks,
CI via GitHub Actions, Cheatsheets.

## Schnellstart
```bash
# 1) Voraussetzung: Flutter SDK, Dart, Java, Android SDK, Melos
dart pub global activate melos
# 2) Repo klonen und bootstrappen
melos bootstrap
# 3) Beispiel-App starten
cd apps/app
flutter run -d android
```

Weitere Docs in `docs/cheatsheets/`.
